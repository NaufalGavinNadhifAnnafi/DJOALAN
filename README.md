## 📌 Tentang Proyek

Djoalan adalah studi kasus desain UX/UI yang mengeksplorasi bagaimana sebuah marketplace mobile dapat menjembatani keaslian budaya pengrajin lokal dengan ekspektasi pembeli modern — menciptakan pengalaman belanja yang premium, terpercaya, dan kaya nilai budaya Indonesia.
